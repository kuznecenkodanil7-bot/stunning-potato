# RM Tools 1.5.0

## Скриншоты наказаний

- Скриншот больше не создаётся поверх окна подтверждения или меню наказаний.
- После успешной команды RM Tools возвращает исходный экран чата и ждёт не менее пяти полностью отрисованных кадров.
- Захват выполняется примерно через 950 мс после закрытия меню, чтобы в кадр попал раскрытый чат и ответ сервера.
- Для нестандартных экранов SmartChat сохранён безопасный резервный таймер.

## HUD

- Минимальный размер увеличен: панель нельзя сжать до нечитаемого состояния.
- Маленький растровый логотип убран из HUD; вместо него используется двухстрочная надпись `RaidMine` / `Tools`.
- Баны, муты и варны отображаются вертикально: иконка строго над числом.
- Верхняя цветная полоска отсутствует.
- Обводка панели включается отдельно и по умолчанию выключена.
- Скругление панели стало ближе к форме капсулы.
- Цвет глаза ваниша меняется сразу при отправке `/v`: фиолетовый — включён, тёмно-серый — выключен.

## Графика и шрифты

- Подключены присланные SVG-иконки, преобразованные в прозрачные текстуры высокого разрешения.
- Атлас иконок и шрифтов использует линейную фильтрацию и увеличенный исходный размер.
- Логотип возвращён к чистому исходнику 1024×1024 без искусственного увеличения и цветных артефактов по краям.
- Добавлены шрифты Segoe UI Variable, Inter, Manrope, Montserrat, Rubik, Bahnschrift, Cascadia Mono, JetBrains Mono, Consolas, Minecraft Pixel и System Mono.
- Вариант Minecraft Pixel использует настоящий встроенный шрифт игры.

## Конфиги

- Основная папка: `config/rm_tools`.
- Настройки: `config/rm_tools/settings.json`.
- Запрещённые слова: `config/rm_tools/forbidden_words.json`.
- Дневная статистика: `config/rm_tools/daily_state.json`.
- Старые файлы автоматически переносятся.
- Для открытия папки больше не используется Java Desktop API; применяются системные команды с резервным копированием пути в буфер обмена.

## SmartChat

- Сначала фиксируются фактические координаты текста из DrawContext — вместе с масштабом, сдвигом и растяжением SmartChat.
- Если сторонний чат не проходит через этот путь, используются координаты, которые выдаёт ChatHud.
- Резервный алгоритм учитывает масштаб и межстрочный интервал SmartChat.
- Кликабельным остаётся только автор сообщения до разделителя, а не упомянутые игроки в тексте.
- Подсветка больше не светится: используется только полупрозрачная заливка и тонкая обводка цветом темы.
