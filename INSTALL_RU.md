# Установка RM Tools 1.7.0 в LabyMod 4

## Требования

- Minecraft 1.21.11
- LabyMod 4 с профилем Fabric
- Java 21

## Сборка и установка

1. Запустите GitHub Actions или `gradle --no-daemon clean build`.
2. Возьмите `build/libs/rm-tools-1.7.0.jar`.
3. Поместите JAR в папку `mods` выбранного Fabric-профиля LabyMod 4.
4. Запустите Minecraft 1.21.11 и подключитесь к RaidMine.

## Проверка

- ЛКМ нажимается только по автору сообщения до `:` или `»`.
- `/rmp Ник` остаётся запасным способом.
- После `/v` глаз в HUD должен сразу стать фиолетовым; повторная команда возвращает тёмно-серый цвет.
- В настройках можно отключить обводку HUD и изменить общую прозрачность фона.

## Скриншоты

После успешного наказания RM Tools закрывает меню, возвращает раскрытый чат, ждёт несколько кадров и сохраняет снимок в `screenshots`:

```text
PlayerName_2.2_2026-07-16_18.30.00.png
```

## Конфиги

```text
config/rm_tools/settings.json
config/rm_tools/forbidden_words.json
config/rm_tools/daily_state.json
```

Файл `forbidden_words.json` можно отправлять другим модераторам. После замены файла нажмите «Обновить» в настройках.
