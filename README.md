# RM Tools 1.7.0

Клиентский мод для персонала RaidMine на **Minecraft 1.21.11 / Fabric / LabyMod 4**.

## Возможности

- ЛКМ по автору сообщения в чате открывает меню наказаний. Упомянутые игроки в тексте не становятся кликабельными.
- Поддержка SmartChat: мод фиксирует фактические координаты текста из DrawContext, затем использует ChatHud и резервный расчёт.
- Ручные команды `/warn`, `/mute`, `/ban`, `/hub` и `/v` учитываются на сетевом уровне, в том числе при отправке через SmartChat.
- Дневное время сохраняется между режимами, останавливается в hub/lobby и сбрасывается в 00:00 по Москве.
- Опциональный AFK Kick отправляет `/hub` ровно после 4 минут бездействия; дневной таймер приостанавливается уже после 1 минуты AFK.
- HUD показывает баны, муты, варны, время и состояние ваниша.
- Настраиваются цвет, прозрачность фона от 0 до 100%, отдельная прозрачность обводок меню/HUD, шрифт, дневная норма и запрещённые слова.
- После наказания меню закрывается, раскрывается чат и только затем создаётся скриншот.
- Каждая иконка загружается отдельной текстурой 1024×1024 с линейной фильтрацией; логотип также использует высокое разрешение.
- AUTO использует Hemico; также доступны Etude Noire и Seenonim. RaidMine Tools всегда отображается Hemico. В проекте нет исходных TTF — используются только сглаженные атласы глифов.

## Управление

- ЛКМ по автору сообщения — открыть наказания.
- `/rmp Ник` — резервное открытие.
- Right Shift — редактор HUD.
- Стрелки — перемещение; Shift+стрелки — быстрый шаг.
- Right Shift или `C` — центрирование; `R` — сброс.
- Маркеры по краям — размер; Ctrl — симметрично относительно центра.

## Конфигурация

```text
config/rm_tools/settings.json
config/rm_tools/forbidden_words.json
config/rm_tools/daily_state.json
```

Старые файлы из `config/rm-tools*` переносятся автоматически. Кнопка «Папка» не использует Java Desktop API: на Windows вызывается Explorer, на macOS `open`, на Linux `xdg-open`; при ошибке путь копируется в буфер обмена.

## Сборка

Требуется Java 21. GitHub Actions уже настроен.

```bash
gradle --no-daemon clean build
```

Результат:

```text
build/libs/rm-tools-1.7.0.jar
```
